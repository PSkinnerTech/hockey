# Convex Backend

This directory contains the Convex backend configuration for Smart Hockey Coach.

## Setup

1. Create a Convex account at https://www.convex.dev/
2. Run `npx convex dev` to start the development server
3. Follow the prompts to create a new project or link to an existing one

## Structure

- `schema.ts` - Database schema definitions
- `_generated/` - Auto-generated TypeScript types (created when you run `convex dev`)
- Feature modules will be added here as the project grows
